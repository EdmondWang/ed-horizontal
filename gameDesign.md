# 游戏主题与需求备忘（持久化）

本文档记录当前已定的 **IP / 设定 / 与布局的对应关系**，供实现与文案对齐；与代码细节以仓库内实际实现为准。

## 已定主题

- **类型**：奇幻 + 环保。
- **对立**：**德鲁伊守护森林家园** vs **兽人大军**（掠夺、开伐、沿一路压境）。
- **气质**：德鲁伊侧强调守护、循环、自然法术；兽人侧强调笨重破坏、断木黑烟，全年龄向，少血腥。

## 一句话设定（文案底子）

> **青林誓环**的德鲁伊守护最后的 **脉心林带**：从 **灵苗军械**（种子、树灵、根须炮）中取出力量，布在 **林缘防线**；**灰斧部族**的兽人伐木军从荒原一侧压境——**林线在，家园在；林线破，根脉断。**

（阵营名、地名可在美术与剧情细化时微调，保持原创。）

## 与主界面三列布局的映射

| 代码中的容器 `label` | 主题化叫法 | 说明 |
|----------------------|------------|------|
| `entityPoolCol` | 灵苗池 / 祭苗栏 | 从池中取单位（或种子）再摆到防线 |
| `defendLineCol` | 林缘防线 / 根缚线 | 德鲁伊与树卫据守；**七纵 × 多列**槽位矩阵（见下「防线槽位」），每格**至多一个单位** |
| `enemyLineCol` | 斧痕荒径 | 兽人兵线推进（右→左）、接敌与射击判定区 |

## 防线槽位与单位类型（规则）

- **槽位**：防线区内为 **7（行，对应车道）× 3（列）** 的格子矩阵；每个格子**独立**，可放置 **0 或 1** 个己方单位。
- **单位大类**（均占用一格，且只能布在己方 `defendLineCol` 范围内）：
  1. **资源采集单位**：随时间或按周期 **产出统一资源**（见下「资源」），自身一般**不主动攻击**或攻击极弱。
  2. **防御单位**：承担输出 / 承伤 / 控制等，**建造与升级需消耗资源**。
- **建造规则**：从 **灵苗池 / 祭苗栏** 选择单位类型 → 点击**空槽**放置；若当前**可用资源不足**对应花费，则不允许放置（需有明确 UI 反馈，如置灰或提示）。

## 资源（经济）

- **统一货币（叙事名）**：例如 **林息** / **脉能**（实现时可用单一整数 `resource` 表示；文案可与「清水、腐殖、日光」等主题资源挂钩，但**数值上先合一**，避免多资源平衡爆炸）。
- **获取**：仅由防线上的 **资源采集单位** 按规则生成（例如每 N 秒 +R，或「收割」触发）；**不得**无代价无限刷兵而不经资源约束（具体数值关卡调参）。
- **消耗**：在空槽部署 **防御单位**、**资源采集单位** 时扣除对应 **花费**；费用为 **非负整数**，填表见下「单位属性 · 数值语义」中的示例占位表。

## 单位属性 · 数值语义

以下适用于 **德鲁伊方** 资源采集单位与防御单位（兽人方可用同一套公式，护甲/血量独立配置）。

| 属性 | 含义 |
|------|------|
| **生命值 HP** | 归零则单位被摧毁 / 撤离（具体由实现定）。 |
| **护甲 Armor** | 参与 **受到伤害** 的减免计算，**不**直接当第二条血条扣减（见下公式）。 |
| **攻击力 Attack** | 每次造成伤害时的 **攻击强度** 输入值；未命中或无攻击行为则为 0。 |

**示例占位（便于程序与关卡填表）**：

| 单位（示例） | 定位 | HP | 护甲 | 攻击 | 部署花费（林息） |
|--------------|------|-----|------|------|------------------|
| 苔泉采集芽 | 资源采集 | 80 | 0 | 0 | 50 |
| 芽弓巡林者 | 基础远程 | 120 | 5 | 22 | 80 |

（上表数值仅为设计文档占位，实现以 `config` 为准。）

## 攻击力 vs 护甲 → 最终伤害（公式）

采用 **乘区减免**：护甲越高，**同一次攻击**的有效伤害越低，但不会出现「护甲 ≥ 攻击力就绝对不破防」的硬阈值，避免前期数值死锁；通过 **最小伤害** 保证低攻仍可有反馈（可选，PvE 可关）。

**定义常数**（建议默认，可在关卡或难度中覆盖）：

- \(K\)：**护甲减伤刻度**，默认 **100**（与百分比直觉对齐：护甲 100 时伤害约为「无护甲时」的一半量级，见下式）。
- \(D_{\min}\)：**最小伤害**，默认 **1**（若希望高甲完全抛光可改为 0）。

**单次结算伤害**（对单次攻击、单个目标）：

\[
\text{最终伤害} = \max\left(D_{\min},\ \left\lfloor \text{攻击力} \times \frac{K}{K + \text{护甲}} \right\rfloor\right)
\]

**说明**：

- 护甲为 **0** 时：\(\lfloor \text{攻击} \times 1 \rfloor = \text{攻击}\)（若攻击为整数则不变）。
- 护甲越大，分数 \(\frac{K}{K+\text{护甲}}\) 越小，伤害平滑下降；**不出现**「攻击力 − 护甲」线性相减导致的负值或一步断层。
- **多段攻击**（如连射）：每一段各自用当前 **攻击力**（可含 buff）代入公式；**DoT** 可另设「每跳攻击强度」同样走此式。
- **兽人 / 环境** 若也有护甲，德鲁伊方打兽人时用 **兽人护甲**；兽人打己方单位时用 **己方护甲**，对称使用同一公式即可。

### 敌人生成权重（可调）

- 每次生成兽人时，在多种基础单位间按 **权重/概率** 抽取（实现为 0～1 常数，便于后期只改编数值）。
- **当前原型约定**：**40%** 为灰斧劫掠兵、**60%** 为投石蛮卒（代码：`SPAWN_WEIGHT_GREY_AXE = 0.4`，其余为投石蛮卒）。

## 胜负条件（叙事包装）

- **胜利**：兽人兵力清零（或关卡目标达成）；文案示例：**「林线还在，根脉未断。」**
- **失败**：兽人冲破林缘 / 母树被伐等；文案示例：**「斧痕过线，母树沉默。」**

玩法层可与现有设计一致：**敌人冲过防御线则败；敌人未过线且敌方数量为 0 则胜**（见历史讨论）。

## 环保主题 · 可落地的机制梗（后续迭代）

- 资源：清水、腐殖、日光等，限制召唤与法术，强调「取用有度」。
- 兽人单位可走「伐木机、燃油罐」等破坏自然意象，避免过度写实血腥。
