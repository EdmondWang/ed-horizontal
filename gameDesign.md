# 游戏主题与需求备忘（持久化）

本文档记录当前已定的 **IP / 设定 / 与布局的对应关系**，供实现与文案对齐；与代码细节以仓库内实际实现为准。

## 已定主题

- **类型**：奇幻 + 环保。
- **对立**：**德鲁伊守护森林家园** vs **兽人大军**（掠夺、开伐、沿一路压境）。
- **气质**：德鲁伊侧强调守护、循环、自然法术；兽人侧强调笨重破坏、断木黑烟，全年龄向，少血腥。

## 一句话设定（文案底子）

> **青林誓环**的德鲁伊守护最后的 **脉心林带**：从 **灵苗军械**（种子、树灵、根须炮）中取出力量，布在 **林缘防线**；**灰斧部族**的兽人伐木军从荒原一侧压境——**林线在，家园在；林线破，根脉断。**

（阵营名、地名可在美术与剧情细化时微调，保持原创。）

## 与主界面三列布局的映射

| 代码中的容器 `label` | 主题化叫法 | 说明 |
|----------------------|------------|------|
| `entityPoolCol` | 灵苗池 / 祭苗栏 | 从池中取单位（或种子）再摆到防线 |
| `defendLineCol` | 林缘防线 / 根缚线 | 德鲁伊与树卫据守；**横向七道防线**，每道**一个槽位**、**一个防御单位** |
| `enemyLineCol` | 斧痕荒径 | 兽人兵线推进（右→左）、接敌与射击判定区 |

## 胜负条件（叙事包装）

- **胜利**：兽人兵力清零（或关卡目标达成）；文案示例：**「林线还在，根脉未断。」**
- **失败**：兽人冲破林缘 / 母树被伐等；文案示例：**「斧痕过线，母树沉默。」**

玩法层可与现有设计一致：**敌人冲过防御线则败；敌人未过线且敌方数量为 0 则胜**（见历史讨论）。

## 环保主题 · 可落地的机制梗（后续迭代）

- 资源：清水、腐殖、日光等，限制召唤与法术，强调「取用有度」。
- 兽人单位可走「伐木机、燃油罐」等破坏自然意象，避免过度写实血腥。

## 与代码模块

- 主界面列容器与背景色：`src/mainLayout.ts`（`mountMainTwoColumnLayout` 返回 `MainLayoutColumns` 供玩法挂载）。
- 逻辑分辨率与画布：`src/core/game.ts`、`src/core/viewport.ts`。
- **最小可玩原型**：`src/game/minimalPrototype.ts`（`mountMinimalForestDefensePrototype`）  
  - **七条横向防线**，每条一个槽；灵苗池显示剩余「芽弓」，点击空槽部署。  
  - **基础远程（芽弓）**仅向**本道防线**上的兽人射击；弹道为**水平线段**（`battleAreaCol` 内绘制，短时淡出）。  
  - 槽位为**明显虚线边框**；兽人从敌线右侧按随机车道生成并左移；冲过敌线左缘则败；生成并清完**较多数量**兽人则胜（见 `ORCS_TO_WIN`）。  
  - 入口：`src/main.ts` 在布局挂载成功后调用。
